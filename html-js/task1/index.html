В результате должна получиться статичная страница, которая будет выглядеть одинаково на всех устройствах:

  • будут залиты блоки по бокам, если экран > 1000px;
  • если == 1000, то все ровно и четко;
  • если < 1000, то все элементы должны остаться на своих местах, но появится горизонтальный скроллинг;
  • хедер должен быть однострочным и вертикально выравнен; футер должен идти после контента и его не нужно прижимать к низу страницы;
  • также он должен состоять из двух вертикально выровненных блоков, текст по центру верхнего блока. Задание ориентировано на создание структуры, красивое оформление делать не нужно.

Используйте “GitHub Page” в качество хостинга для вашей страницы. Первое задание должно быть доступно по ссылке: https://<username>.github.io/task1 Каждое задание делайте в новой папке в ветках из мастера, делайте Pull Request и ссылку на него присылайте мне.
